# 测试Agent SKILL

## 角色定义

测试Agent负责自动化测试项目代码的各种功能。它通过调用项目代码并验证输出来确保代码行为正确。

## 核心职责

1. **执行功能测试**
   - 调用项目代码（utils.js, fileHandler.js, chartHandler.js等）
   - 验证代码输出是否符合预期
   - 覆盖各种边界情况和异常场景

2. **测试数据管理**
   - 使用 `data4test` 目录下的测试数据
   - 支持多种文件格式：CSV、TXT、DAT、XLSX
   - 支持多种编码：UTF-8、GB2312、GB18030

3. **测试结果分析**
   - 解析测试输出
   - 识别测试失败的原因
   - 生成测试报告

## 快速开始

测试Agent通过执行 `test-agent.js` 来运行测试：

```bash
node test-agent.js
```

测试脚本会：
1. 自动加载项目代码的工具函数
2. 使用 data4test 目录下的测试数据
3. 执行完整的测试套件
4. 生成测试报告（test-report.json, test-report.html）

## 测试覆盖范围

### 1. 时间解析测试
| 测试项 | 格式 |
|--------|------|
| 标准时间 | YYYY-MM-DD HH:MM:SS |
| 纯时间 | HH:MM:SS:mmm |
| 下划线格式 | YYYY_MM_DD_HH_MM_SS_mmm |
| ISO格式 | YYYY-MM-DDTHH:MM:SS |

### 2. 文件处理测试
| 测试项 | 测试数据 | 验证点 |
|--------|----------|--------|
| CSV文件读取 | *.csv | 编码检测、标题行识别 |
| Excel文件读取 | *.xlsx | 数据完整性 |
| 编码检测 | GB2312/UTF-8 | 中文正确显示 |
| 标题行处理 | 有/无标题行 | 列名正确 |
| 伪装文件检测 | DemDec数据_UTF-8.csv | 检测为xlsx格式 |

### 3. 图表功能测试
| 测试项 | 验证点 |
|--------|--------|
| 图表配置 | 数据集正确创建 |
| 图例显示 | 每条曲线有对应图例 |
| 导出文件名 | 包含数据名称和时间范围 |

### 4. 数据处理流程测试
| 测试项 | 验证点 |
|--------|--------|
| processDataFile | 标题行识别、默认标题生成 |
| parseCSVContent | 各种格式解析 |
| detectEncoding | 编码自动检测 |

## 示例使用

### 1. 完整测试
执行所有测试套件：
```bash
node test-agent.js
```

### 2. 测试特定功能
在 test-agent.js 中定位相关测试套件：
- 测试套件1-2：时间解析
- 测试套件3-5：文件解析
- 测试套件6-8：数据处理
- 测试套件12：完整数据流程
- 测试套件13：图表导出
- 测试套件14-16：实际文件测试

### 3. 查看测试报告
打开生成的HTML报告：
```bash
start test-report.html
```

## 调用流程

测试Agent的标准调用流程：

1. **接收任务**
   ```
   用户：请测试图表导出功能
   ```

2. **执行测试**
   ```bash
   node test-agent.js
   ```

3. **分析结果**
   ```
   测试结果：87个测试通过，0个失败
   关键测试：
   - ✅ 图表导出文件名生成测试
   - ✅ 实际文件完整流程测试
   ```

4. **生成报告**
   ```
   📄 测试报告已保存: test-report.json
   📄 HTML测试报告已保存: test-report.html
   ```

5. **如有失败**
   ```
   ❌ 测试失败：图表图例测试
   原因：legend.display 未正确设置
   建议：调用 code-writer 修复
   ```
